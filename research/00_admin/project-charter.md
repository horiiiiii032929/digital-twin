# Project Charter

## Objective

Build and evaluate a Digital Twin teaching system that can answer course-bound
student questions in an instructor-aligned teaching style and summarize learning
gaps for the instructor.

## Scope

- Instructor onboarding and teaching-policy capture
- Course material ingestion and RAG baseline
- Student tutoring prototype
- Learning-gap reporting prototype
- Evaluation against baseline assistant behavior

## Out of Scope For Initial Scaffold

- Production deployment
- Institutional LMS integration
- Real student data ingestion without explicit consent and review
